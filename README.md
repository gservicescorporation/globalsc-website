🌐 Global Services Corporation — Website Oficial

Este é o repositório do site institucional da Global Services Corporation, desenvolvido com Next.js
 e outras tecnologias modernas para garantir performance, escalabilidade e excelente experiência do utilizador.

🚀 Tecnologias Principais

Next.js 15 – Framework React de última geração

TypeScript – Tipagem estática para mais segurança e escalabilidade

Tailwind CSS – Estilização rápida e responsiva

Axios – Consumo de APIs externas

Framer Motion – Animações fluidas e modernas

Turbopack – Compilação ultra rápida para produção

🛠️ Como executar o projeto

Clone o repositório e instale as dependências:

git clone https://github.com/SEU_USUARIO/globalsc-website-v2.git
cd globalsc-website-v2
yarn install


Inicie o servidor de desenvolvimento:

yarn dev


O site estará disponível em:
👉 http://localhost:3000

🧩 Estrutura principal do projeto
globalsc-website-v2/
├── app/               # Páginas e rotas principais
├── components/        # Componentes reutilizáveis
├── api/               # Configuração de APIs
├── public/            # Imagens e arquivos estáticos
├── styles/            # Estilos globais
└── package.json       # Configurações e dependências

✏️ Personalização

As páginas podem ser editadas diretamente em:

app/page.tsx


O projeto usa next/font
 para otimizar e carregar a fonte Geist
, garantindo performance e estética consistente.
